Happy OSO 🐶
A Flappy Bird-style iOS game starring Oso — a real-life Shih Tzu — built with Swift and SwiftUI.

📖 About
Happy OSO is a fun, personal iOS game created as a gift for Amy and Oso. Guide Oso through a series of tree obstacles in a beautiful nature-themed world. Tap to flap, dodge the trees, and beat your high score!

🎮 Gameplay

Tap the screen to make Oso fly upward
Dodge the tree trunk obstacles
Score points every time you pass through a gap
Beat your best — your high score is tracked every session

🛠 Built With

Swift — Apple's programming language
SwiftUI — Declarative UI framework for iOS
Combine — For game loop timers and reactive state
No third-party libraries — 100% native iOS


📱 Requirements

iOS 16.0 or later
Xcode 14 or later
iPhone (optimized for modern screen sizes)


🚀 Running the App

Clone the repo:

bashgit clone https://github.com/jozuna19/HappyOso.git

Open the project in Xcode
Add your own oso.png image to Assets.xcassets (or use any transparent PNG)
Select a simulator or your iPhone as the target
Hit ▶️ Run


✨ Features

🐶 Real photo of Oso as the playable character
🌲 Nature-themed background with scrolling clouds, rolling hills, and a full forest
🏆 Persistent score counter and high score tracker
🌸 Smooth physics with forgiving hitbox
🎉 Animated game over and start screens
